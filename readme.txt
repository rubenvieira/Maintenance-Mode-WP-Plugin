== Description ==

A landing page plugin with a simple on/off toggle in the admin bar.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/maintenance-mode` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the toggle in the admin bar to enable/disable maintenance mode.
